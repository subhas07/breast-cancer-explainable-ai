# 📌 Project Overview

The objective of this project is to build an intelligent healthcare prediction system that can classify tumors as:

- **Benign (Non-Cancerous)**
- **Malignant (Cancerous)**

using Machine Learning algorithms and provide explainable predictions through SHAP Explainable AI.

---

# 🧠 Explainable AI Integration

This project integrates **SHAP (SHapley Additive exPlanations)** to improve transparency and interpretability of predictions.

SHAP helps explain:
- Which features influenced the prediction
- Positive/negative feature contributions
- Model decision behavior

This is especially important in healthcare AI applications.

---

# 📊 Features Used

The model uses important medical diagnostic features such as:

- Radius Mean
- Texture Mean
- Perimeter Mean
- Area Mean
- Smoothness Mean
- Compactness Mean
- Concavity Mean
- Concave Points Mean
- Symmetry Mean
- Fractal Dimension Mean
